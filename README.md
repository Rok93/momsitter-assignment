- [ ] API 스펙 정의
- [ ] ERD 설계
- [ ] 라이브러리를 사용한 이유 명시

        * 비즈니스 코드를 더럽히는 Swagger의 장점이 너무 싫어서 RestDocs를 사용
- [x] 비밀번호 정책
    * (8자리 ~ 16자까지 허용, (숫자, 문자, 특수문자)가 포함되도록 패턴 지정
